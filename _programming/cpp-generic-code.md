*Effective Modern C++ Chapter 3*

* In maximally generic code, prefer non-member versions of `begin`, `end`, `cbegin`, etc., over their member function counterparts.
   * 所有就能理解为什么会有那么多non-member functions了。
   * std::to_string
   * std::swap, std::sort
