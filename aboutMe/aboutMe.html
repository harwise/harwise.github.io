[VMWare] Horizon远程桌面重定向功能

项目简介

VMWare Horizon远程桌面系统中，实现在远程桌面和本地客户端之间，对设备，数据，或者用户操作进行重定向，让使用远程桌面更加方便。

所需技能

C++, DirectShow, USB, HID, NBD, Scanner(TWAIN, SANE)

贡献

主导完成Windows Media Player重定向功能的开发：把远程桌面端WMP的视音频数据重定向到Client端解码和渲染。通过实现DirectShow的COM组件来接管视音频数据和播放控制指令。

对USB重定向功能中的HID设备进行改进：提高业务数据的传输效率，以解决网络延迟引起的HID设备数据丢失。针对HID的特性优化USB数据包的管理和传送，提高了远程桌面端的对设备请求的响应速度。

实现对磁盘分区的重定向：让本地客户端的磁盘能够被远程桌面直接读写。使用NBD（Network Block Device）技术实现，工作在文件系统层之下的存储块设备层。

实现扫描仪的重定向：依据TWAIN规范，在远程桌面端实现一个虚拟扫描仪，映射到本地客户端的物理设备。

实现扫描仪的重定向：让Linux客户端的扫描仪能够被远程桌面直接使用。依据SANE规范控制物理扫描仪，并转换成远程桌面识别的TWAIN协议数据。

[VMWare] Horizon Remote Experience

Summary

Local devices are redirected to remote desktops, providing seamless user experience。

Skills

C++, DirectShow, USB, HID, NBD, Scanner(TWAIN, SANE)

Contributions

Be responsible for the development of Windows Media Player Redirection. WMP video/audio streams are sent from remote desktops to Clients, decoded and rendered. By implementing DirectShow video/audio filters, video/audio streams and user commands on remote desktops can be intercepted.

Improved the HID device performance for USB Redirection. Data losses caused by network latency is resolved. By optimizing the management and transmit of HID packets, virtual devices can respond in time on remote desktops.

Scanner Redirection. By following the TWAIN Protocol on Windows, a virtual scanner on remote desktops was implemented.

Scanner Redirection. On Linux systems, following the SANE protocol, local devices are controlled and made accessble on remote desktops.
