[VMWare] Horizon远程桌面重定向功能

项目简介

VMWare Horizon远程桌面系统中，实现在Agent和Client之间，对设备，数据，或者用户操作进行重定向，让使用远程桌面更加方便。

所需技能

C++, DirectShow, USB, HID, NBD, Scanner(TWAIN, ICA)

贡献

主导完成Windows Media Player重定向功能的开发：把Agent端WMP的视音频数据重定向到Client端解码和渲染。通过实现DirectShow的COM组件来接管视音频数据和播放控制指令。

对USB重定向功能中的HID设备进行改进：提高业务数据的传输效率，以解决网络延迟引起的HID设备数据丢失。针对HID的特性优化USB数据包的管理和传送，提高了Agent端的对设备请求的响应速度。

实现对磁盘分区的重定向：让Client端的磁盘能够被Agent直接读写。使用NBD（Network Block Device）技术实现，工作在文件系统层之下的存储块设备层。

实现扫描仪的重定向：让Client端的扫描仪能够被Agent直接使用。依据TWAIN规范和MAC上ICA规范进行开发，在Agent端实现一个虚拟扫描仪。

[VMWare] Horizon Remote Experience

Summary

For VMWare Horizon Remote Desktop, Specific devices, data and user actions are redirected between Agent and Client, providing seamless user experience。

Skills

C++, DirectShow, USB, HID, NBD, Scanner(TWAIN, ICA)

Contributions

Be responsible for the development of Windows Media Player Redirection. WMP video/audio streams are redirected from Agent to Client, and decoded and rendered on the latter. By implementing DirectShow video/audio filters, video/audio streams and user actions on Agent can be intercepted.

Implemented Scanner Redirection. This feature makes Client scanner devices be used on Agent. By following TWAIN and ICA Protocol, a virtual scanner on Agent was implemented.
